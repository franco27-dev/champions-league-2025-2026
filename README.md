# champions-league-2025-2026
Análisis completo de la Champions League 2025/2026 con 144 partidos y 487 goles. Incluye extracción y consultas en SQL (SQLite), limpieza y visualización en Python (pandas, matplotlib, seaborn), tablas y gráficos dinámicos en Excel, y dashboard interactivo en Power BI con KPIs, gráfico de barras y gráfico circular.
